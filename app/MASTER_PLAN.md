# Master Plan

## Phase 1 — Safe Inventory
- Scan selected FATLETIC root directory.
- Do not move, rename, delete, or modify originals.
- Create full file manifest.
- Hash every file.
- Extract basic metadata.

## Phase 2 — Classification
- Classify files into asset types.
- Assign confidence scores.
- Flag low-confidence items for review.
- Detect duplicates and near-duplicates.

## Phase 3 — Evidence Modeling
- Build relationships between assets.
- Connect design files to exports, mockups, product orders, social posts, and customer photos.
- Build timeline events with proof links.

## Phase 4 — Archive Generation
- Create organized archive copy.
- Preserve originals in read-only source mode.
- Generate documentation and indexes.

## Phase 5 — Dashboard
- Build local dashboard for timeline, evidence, reports, search, review queue, archive health, and export packages.

## Phase 6 — Export Packages
- Priority of Use Dossier.
- Trademark Readiness Report.
- Evidence Binder PDF.
- Attorney Review ZIP.
- Investor/Brand History Pack.
