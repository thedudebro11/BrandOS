# BrandOS — Application

This folder contains the BrandOS application: specs, docs, prompts, and (eventually) source code for the multi-workspace evidence and brand-archive platform. See the repository root `README.md` for the platform overview.

Goal: preserve originals, organize evidence, classify every asset, generate timelines, build cases, create legal-quality reports, and make each workspace's history searchable and reviewable — for any workspace, not only Fatletic (the first workspace, see `workspaces/Fatletic/`).

This system is not legal advice and does not decide trademark or legal rights. It creates a neutral, evidence-backed archive that can support attorney review, filings, due diligence, and long-term brand or knowledge management.

## Structure
- `specs/` — numbered core platform specs (`00`–`21`) plus `specs/plugins/` for brand/source-specific plugin specs.
- `docs/` — architecture, implementation plan, risks, open questions, ADRs, changelog.
- `prompts/` — build prompts.
- `src/`, `plugins/`, `shared/`, `scripts/`, `tests/` — application code (not yet written).
