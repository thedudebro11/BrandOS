# File Scanner

## Requirements
- Recursively scan the selected workspace's evidence directory.
- Record every file and folder.
- Capture path, filename, extension, size, created date, modified date, accessed date if available.
- Detect hidden files.
- Detect broken shortcuts.
- Detect empty folders.
- Detect suspicious duplicates.

## Output
- FILE_MANIFEST.csv
- FILE_MANIFEST.json
- Directory_Index.md

## Safety Rule
Never move or rename files during scan mode.
