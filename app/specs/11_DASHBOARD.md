# Dashboard Specification

## Yes, There Should Be a Dashboard
The organized directory is the storage layer. The dashboard is how the user sees the story, evidence, risks, and progress.

## Dashboard Pages
1. Home / Archive Health
2. Master Timeline
3. Priority of Use Dossier
4. Trademark Readiness Report
5. Evidence Graph
6. File Explorer
7. Human Review Queue
8. Evidence Gaps
9. Logo Evolution
10. Product History
11. Social Media Archive
12. Export Center

## Home Metrics
- Total files scanned
- Files classified
- Files needing review
- Duplicate groups found
- Earliest FATLETIC evidence date
- Earliest commercial use date
- Trademark readiness score
- Priority of use confidence score

## Visuals
- Timeline cards
- Evidence score gauges
- File preview panel
- Relationship graph
- Missing evidence checklist
- Export package builder
