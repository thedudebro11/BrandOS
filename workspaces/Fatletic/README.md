# Fatletic Workspace

This workspace contains the business, brand, design, social, sales, trademark, copyright, and historical evidence for the Fatletic brand.

## Important Rule

Do not delete, overwrite, rename, or modify original evidence files without explicit approval.

Original files should be preserved, hashed, indexed, and referenced before any organized archive views are created.

## Workspace Purpose

Fatletic is the first workspace inside BrandOS. The goal is to preserve and organize the complete history of the Fatletic brand, including:

- brand development
- logo evolution
- product development
- Printful orders
- Instagram history
- customer evidence
- trademark evidence
- copyright evidence
- priority of use evidence
- business milestones

## Key Legal/Evidence Focus

The primary evidence focus is supporting the history and use of the FATLETIC brand, including potential common law trademark rights, federal trademark readiness, priority of use, and supporting documentation.

Known related/conflicting mark for research:

- FATLETE

## Evidence Handling

Every original file should eventually receive:

- SHA-256 hash
- original path
- file metadata
- classification
- confidence score
- relationship links
- timeline references
- evidence relevance
- Obsidian note reference

## Case Builder

The workspace should support named cases such as:

- Trademark Registration
- Trademark Opposition
- Copyright Registration
- Investor Due Diligence
- Brand Acquisition
- Media Kit
- Product Launch
- Historical Timeline

Cases should reference files, not duplicate them.
